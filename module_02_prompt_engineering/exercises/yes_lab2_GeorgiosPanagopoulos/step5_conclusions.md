# Συμπεράσματα Βήμα 5

## Πείραμα 1: Σύγκριση Personalities
Στέλνοντας την ίδια ερώτηση σε 3 personalities, πήραμε 3 τελείως διαφορετικές απαντήσεις. Ο πειρατής μιλούσε με "Arr" και ναυτικές μεταφορές, ο cynical agent με σαρκασμό, ο zen master με φιλοσοφία. Οι πληροφορίες όμως ήταν σωστές και στις 3 περιπτώσεις. Δηλαδή το system prompt αλλάζει το ύφος, όχι την ακρίβεια.

## Πείραμα 2: Temperature
- 0.0 → Οι 3 απαντήσεις βγήκαν σχεδόν ίδιες
- 0.5 → Μικρές αλλαγές στη διατύπωση
- 1.0 → Κάθε απάντηση ήταν διαφορετική

Όσο ανεβαίνει η temperature, τόσο πιο απρόβλεπτο και δημιουργικό γίνεται το μοντέλο.

## Συμπέρασμα
Με system prompt + temperature ελέγχεις πλήρως πώς απαντάει ένα LLM, χωρίς να χρειάζεται fine-tuning.
